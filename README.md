# Getting Started

#### Development 

1. Clone repo
     ```bash
      $ git clone https://github.com/Himanshuraj2918/Voting_App---Nodejs.git
		
	    $ cd Voting_App---Nodejs

	    # Remove current origin repo
	    $ git remote remove origin  
     ````
2. Start Server

   ```bash
    $ cd api
    $ npm install
    $ npm start
   ```
